# Razor Pages Practice Website

**Author:** Joe Goode

This project is a simple ASP.NET Core Razor Pages website created as a practice activity for learning Razor Pages and basic page structure.

---

## Pages Included

The website contains the following pages:

- **Home**
- **About**
- **Contact**

---

## Home Page

The Home page includes:

- A title
- A short welcome message

---

## About Page

The About page contains brief information about the author.

---

## Contact Page

The Contact page includes a basic contact form with the following fields:

- Name
- Email
- Message
- Submit button

The form is for practice purposes only and does not send emails.

---

## Technologies Used

- ASP.NET Core Razor Pages
- Bootstrap (default template styling)
- Visual Studio 2022

---

## How to Run

1. Open the project in Visual Studio.
2. Press **F5** to run the application.
3. The site will open locally in your browser.

---

## Notes

This project focuses on Razor Pages structure rather than advanced styling or backend processing.
